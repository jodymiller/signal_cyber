#Scripting and automation

Python is such a versatile programming language and bash is incredibily useful when working in Linux. 

My scripts are experimental, learning territory to better understand the tools available and how to utilize them in any given system. Some and may even mimic some long standing security tools; however, the point is to write them myself (and co-write with AI tools to check myself), tweak them through trial and error, and find new ways of doing things (maybe). There likely is no revelation here, but maybe someone will find one or more of these useful.
